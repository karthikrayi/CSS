CSS Position Property

An overview of the different position values in CSS.

Static Position

The default position value for HTML elements.

Static Box

Relative Position

The element is positioned relative to its normal position.

Relative Box

Absolute Position

The element is positioned absolutely to its closest positioned ancestor.

Absolute Box

Fixed Position

The element is positioned relative to the viewport and stays fixed when scrolling.

Fixed Box

Sticky Position

The element toggles between relative and fixed, depending on the scroll position.

Sticky Box
